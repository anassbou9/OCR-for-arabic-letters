# OCR-for-arabic-letters

In order to run the scripts, go to src_code directory.

-Running segmentation.py script will segment lines and words and put them respectively in lines and words directories


-Running char_seg.py will segment a word image into characters and will put them in chars directory


-Running predict_word.py will print the word given as input


-Running post_processing.py prints a processed text that is given as input


-The main script is ocr.py, it prints the predicted text.

The input is given in the beginning of every script in the INPUT_TEST_PATH variable
